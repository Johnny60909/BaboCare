# 🍼 BaboCare：居家托育安心動態系統
> **讓托育紀錄像發 IG 一樣簡單有趣，讓家長參與寶寶的每一秒成長。**

---

## 🌟 專案願景 (Vision)
在居家保母的工作中，**「信任」**是核心價值。**BaboCare** 重新定義了保母與家長的溝通方式。我們捨棄了枯燥的紙本紀錄與雜亂的 LINE 群組，轉而建立一個**專屬於寶寶的微型社群**。

保母能以最低的操作成本（一鍵紀錄）完成繁瑣的行政庶務；家長則能隨時透過手機，像刷社交媒體一樣，即時看見寶寶的現況、照片與心情，有效消除分離焦慮。

---

## ✨ 核心價值 (Core Values)

* **🛡️ 建立信任：** 透過即時動態牆，打破保母與家長間的資訊不對稱。
* **⚡ 極簡操作：** 針對保母忙碌的環境設計「大按鈕主控台」，單手即可完成紀錄。
* **📱 沉浸體驗：** 採用類 Instagram 的 UI 設計，讓看紀錄變成一種療癒的享受。
* **🗓️ 資訊對齊：** RWD 雙向行事曆，確保接送、餵藥與收費資訊透明化。

---

## 🛠️ 技術架構 (Technical Stack)
*本專案為個人開發作品，採用SDD openspec AI開發方式，全程以AI Coding為基礎目標*

* **Backend:** `.NET 10 (C#)` - Clean Architecture、DDD。
* **Database:** `PostgreSQL 10` - 採用 EF Core 10 。
* **Frontend:** `Vue.js` - 搭配 UI框架 PrimeVue。
