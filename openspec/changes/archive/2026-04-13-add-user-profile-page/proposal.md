# Add User Profile Page

## What

在首页的底部导航栏最右边添加**个人资料中心**的图标按钮，点击后进入一个新的个人资料页面。该页面包含：

- 顶部导航栏（与前台风格一致）
- 右上角有一个"三个点"设置图标
- 点击图标会弹出下拉菜单，底部有红色**登出**按钮
- 中间内容区域留白，用于后续扩展

## Why

- **用户体验**：提供快速访问个人资料和设置的入口
- **功能完整性**：实现登出功能，用户能够安全退出系统
- **UI 一致性**：与现有的 iOS 风格导航栏保持统一
- **可扩展性**：为后续的账户设置、个人信息编辑等功能预留空间

## Scope

### Included

- 前端：在导航栏添加个人资料图标
- 前端：创建个人资料页面（`/profile`）
- 前端：实现顶部设置菜单（下拉菜单 + 登出按钮）
- 前端：集成登出 API 调用

### Not Included

- 个人信息编辑功能
- 账户设置详情页面
- 后端新增 API（使用现有登出接口）

## Acceptance Criteria

- [x] 底部导航栏显示个人资料图标（最右侧，在返回首页按钮之后）
- [x] 点击图标进入 `/profile` 页面
- [x] 页面顶部有设置图标
- [x] 点击设置图标显示下拉菜单，包含登出按钮
- [x] 登出按钮样式为红色，点击后清空认证信息并跳转到登录页
- [x] 整体风格与前台保持一致（iOS 卡片风格、圆角按钮等）
